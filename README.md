Digital Image Processing Lab, CSE 4228
--------------------------------------------------------
IDE: Matlab
--------------------------------------------------------
Features
--------------------------------------------------------
1.	Pupil localization
2.	Pupil center detection
3.	Marking the pupil in the image
4.	Eye gazing detection of the following directions:
      a.	Left
      b.	Right
      c.	Up
      d.	Down
